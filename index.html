Prácticas de Programación Web - Unidad 2

Menú de proyectos realizados en la Unidad 2 de Programación Web

En esta unidad exploramos los fundamentos de HTML y CSS mediante diversas prácticas diseñadas para desarrollar nuestras habilidades en la creación de páginas web estructuradas y visualmente atractivas.

  • Currículum Vitae en HTML: Aprendimos la estructura básica de una página web creando nuestro propio CV, utilizando etiquetas semánticas para organizar la información de manera clara y accesible.
  • Historia de las Aplicaciones Web: Construimos una página interactiva con anclas para facilitar la navegación entre secciones, repasando la evolución de las aplicaciones web.
  • Perfil Profesional: Desarrollamos un blog personal con elementos multimedia, aplicando conceptos de diseño y presentación de contenido profesional.
  • Galeria de ciudades: Creamos una galería interactiva mostrando las ciudades que deseamos visitar, utilizando CSS para estilizar las imágenes y GitHub Pages como plataforma de publicación y prueba.
  • Guía Turística de Yanga: Diseñamos una página web con información sobre nuestra ciudad, aplicando CSS para mejorar la estética y la experiencia del usuario.
